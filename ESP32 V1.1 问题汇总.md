进入了烧录模式

ESP32-C3数据手册：

| 参数    | 名称         | 含义                                               | ESP32-C3值  |
| ----- | ---------- | ------------------------------------------------ | ---------- |
| `tSU` | Setup time | 电源稳定后，到 `CHIP_EN` 拉高前，给 strapping pin 准备状态的时间    | ≥ 0 ms     |
| `tH`  | Hold time  | `CHIP_EN` 拉高后，芯片读取 GPIO2/GPIO8/GPIO9 状态，并保持采样的时间 | **≥ 3 ms** |
Strapping Pins：GPIO2/GPIO8/GPIO9

结果：EN延时电路+GPIO9上拉电阻
