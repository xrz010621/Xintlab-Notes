# MT6701传感器精度测试方案文档

>类型：agent生成初步分析报告

**磁铁尺寸/间距对 MT6701 霍尔传感器精度影响测试**

**目的**

在 EVT v2 电机模块上测试不同直径、厚度的径向充磁圆柱形磁铁对 MT6701 霍尔传感器精度的影响。

- 调整磁铁与传感器间距，分别验证 2mm / 3mm / 4mm / 5mm 的可用性及磁铁尺寸适配性。

- 在电机单元转动范围内评估精度影响。



**对象**

MT6701——14位绝对式磁旋转编码器。



**原理**

磁铁的直径、厚度、气隙直接影响霍尔传感器接收到的磁场强度，当前项目固件读取MT6701的两个角度寄存器，因此判断磁铁质量的间接指标即角度数据。



**数据链路拓扑**

┌──────────────┐     UART 115200      ┌──────────────────┐     软件模拟 I²C      ┌──────────┐
│   PC (脚本)   │ ◄──────────────────► │  APM32F103CBT6   │ ◄──────────────────► │  MT6701  │
│  pyserial    │   ASCII 命令/回复     │  test_shell.c    │   PB10(SDA)/PB11(SCL) │  编码器   │
└──────────────┘                      └──────────────────┘                      └──────────┘


<u>流程</u>

PC 脚本 → UART → 发 `STATUS` 命令 → APM32 固件内部调用 `Mt6701ReadAngleRaw()` 通过软件 I²C 读取 MT6701 → 将 `raw` 和 `angle_x100` 打包成 `K=V` 文本 → UART → 返回给 PC 脚本



**引脚**

APM32的UART引脚：PA9(TX) | PA10(RX), 115200, 8N1

MT6701通过GPIO模拟的I2C挂在APM32的PB10(SDA)/PB11(SCL)上

MT6701 的 I²C 地址为 0x06（7位），角度数据分布在两个寄存器：0x03（ANGLE_H，高8位）和 0x04（ANGLE_L，低6位在 [7:2]）



**方案**

**采样点**

编码器转动角度接收范围为\[1\.5°,71\.5°\]，共70°行程。以5°为步长，可以得到共15个均匀分布的采样位置。

通过固件的数据返回，可以得到raw、angle\_x100。



**测量流程：**

**\[逐点采集\] 对每个目标角度angle\_i ∈\[1\.5°, 6\.5°，11\.5°,…, 66\.5, 71\.5°\]**

1a\. 通过MOVE \<angle\_i \*100\> 使电机转动到当前测试角度。

1b\. 直到status状态为arrived，重复读取raw值，每个角度采集50帧中每帧的raw，angle\_x100。

1c\. 计算该点的均值、标准差σ、峰峰值

2\. 到下一个采样角度，重复数据采集过程\(1a\~1c\)。

**\[间距递增\]**

3\. 同一磁铁调整间距至2mm/3mm/4mm/5mm，各重复1a\~2。

**\[磁铁更换\]**

4\. 更换磁铁，重复1a\~3的测量步骤。



**数据结果**

1. 间距（2mm/3mm/4mm/5mm）\&磁铁规格（D×T）矩阵表格，每格包含角度采样点中最大σ值。

2. 每种磁铁规格的详细数据：4个间距下不同角度采样点的均值、σ值、峰峰值，并绘制对应的\[σ\-angle\]曲线。
