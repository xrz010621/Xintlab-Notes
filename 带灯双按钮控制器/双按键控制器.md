## 状态定义

| State    | Description             |
| -------- | ----------------------- |
| IDLE     | Waiting for user input  |
| RUNNING  | Executing selected mode |
| FINISHED | Task completed          |
| ERROR    | Fault state             |

## 状态机示意图  

```mermaid
stateDiagram-v2 

state RUNNING{
	[*] --> RETURN_MODE
	RETURN_MODE --> RANDOM_MODE: Switch Mode
	RANDOM_MODE --> RETURN_MODE: Switch Mode
}

IDLE --> RUNNING: Press
RUNNING --> FINISHED: Complete
RUNNING --> ERROR: Error
RUNNING --> IDLE: Stop
IDLE --> ERROR: Error
ERROR --> IDLE: Recover
FINISHED --> IDLE

```
