# 回球路线测试

# 1目标

在不同地形下，都能实现回球

# 2测试设备

样机\*1

卷尺

测试APP

# 3测试步骤

1\.将样机设定为目标地形，用卷尺测量电机高度，保证电机高度为理论角度高度

2\.将1个或多个球放置于球洞处

3\.抬起34电机，降低56电机，将球从球洞处倒出

4\.选择56电机其一抬起，降低1234电机至水平位置，使球滚至一侧

5\.抬起56电机，使球回滚
